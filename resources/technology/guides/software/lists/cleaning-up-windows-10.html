Cleaning-up Windows 10

Collecting Tools

  1. Visit the following links on the target system and download the portable executable archives. Extract and place the folders on the desktop
    • BleachBit | FOSS System Cleaner
    • PrivaZer | System Cleaner
    • CCleaner | System Cleaner
    • Device Cleanup Tool | Windows Device History Remover

Windows Techniques 1

User Folders
  1. Hit "Win + E" on your keyboard to open an Explorer window
  2. Delete old or unnecessary files from "C:\", "Desktop", "Documents", "Downloads", "Music", "Pictures" and "Videos"
  3. Exit
Existing Applications
  1. Start Menu > Settings > Apps > Apps & Features
  2. Uninstall any apps no longer required
  3. Exit and restart
Start Menu
  1. Hit "Win + E" on your keyboard to open an Explorer window
  2. Copy and paste C:\Users\%username%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs into the address bar, then press the enter key
  3. Move application shortcuts located in a folder into the "Programs" folder (e.g., "MPC-HC x64" in the "MPC-HC x64" folder should be moved directly into C:\Users\%username%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs and "Uninstall MPC-HC x64" should be left in the "MPC-HC x64" folder)
  4. Delete any residual folders (e.g., the "MPC-HC x64" folder should be deleted)
  5. Repeat with C:\ProgramData\Microsoft\Windows\Start Menu\Programs
  6. Exit
Program Files
  1. Hit "Win + E" on your keyboard to open an Explorer window
  2. Copy and paste C:\Program Files into the address bar, then press the enter key
  3. Delete any folders related to uninstalled applications
  4. Repeat for C:\Program Files (x86)
  5. Exit
Program Data
  1. Hit "Win + E" on your keyboard to open an Explorer window
  2. Type "%programdata% into the address bar, then press the enter key
  3. Delete any folders related to uninstalled applications
  4. Exit
App Data
  1. Hit "Win + E" on your keyboard to open an Explorer window
  2. Copy and paste C:\Users\%username%\AppData\Roaming into the address bar, then press the enter key
  3. Delete any folders related to uninstalled applications
  4. Copy and paste C:\Users\%username%\AppData\Local into the address bar, then press the enter key
  5. Delete any folders related to uninstalled applications, and any "cache" folders
  6. Exit
Registry
  1. Hit "Win + R" on your keyboard, type "regedit", then press the enter key
  2. Copy and paste Computer\HKEY_CURRENT_USER\SOFTWARE into the address bar, then press the enter key
  3. Hit "Win + R" on your keyboard, type "regedit", then press the enter key
  4. Delete any folders related to uninstalled applications
  5. Repeat for Computer\HKEY_LOCAL_MACHINE\SOFTWARE
  6. Exit
Temp
  1. Hit "Win + E" on your keyboard to open an Explorer window
  2. Type "%temp%" into the address bar, then press the enter key
  3. Hit "Ctrl + A" on your keyboard to select all files and folders
  4. Hit "Del" on your keyboard
  5. If prompted to provide administrator permission, check "Do this for all current items" and click "Continue"
  6. If an action cannot be completed, check "Do this for all current items" and click "Skip"
  7. Exit

Using Tools

Device Cleanup
  1. In the "tools" folder, right-click "DeviceCleanup" then click "Run as administrator"
  2. Hit "Ctrl + A" on your keyboard to select all items
  3. Devices > click "Remove selected"
  4. Exit
CCleaner
  1. Open "CCleaner64" from the "tools" folder
  2. Registry > click "Scan for Issues"
  3. Click "Review selected Issues...", then "No", then "Fix All Selected Issues"
  4. Repeat steps 2 and 3 once more
  5. Exit
PrivaZer
  1. Open "PrivaZer" from the "tools" folder
  2. Click "Cancel" then click "Basic user" to change to "Advanced user"
  3. Click "Advanced Options" > Memory > check "After each cleanup with PrivaZer" then click "OK"
  4. Click the blue arrow to return
  5. Click "Scan in-depth" then click "Computer"
  6. Check all "Select scans" options
  7. Click "Modify", click "OK" and change "Smart" to "Remove all"
  8. Click the blue arrow to return
  9. Click "Scan"
  10. Click "Clean" then "Normal cleanup"
  11. Click "OK", "Close" then exit
BleachBit
  1. Open "bleachbit" from the "tools" folder
  2. Click the icon in the top left, then click "Preferences"
  3. Check "Download and update cleaners from community (winapp2.ini)" then click "Close"
  4. Exit and reopen "bleachbit"
  5. Check all folded categories and click "OK" to all prompts/li>
  6. Uncheck "Free disk space" under "System"
  7. Click "Clean" then "Delete"
  8. Exit

Windows Techniques 2

Disk Clean-up
  1. Delete all tools
  2. Start Menu > Windows Administrative Tools > right-click "More" > click "Run as administrator"
  3. Check all options under "Files to delete:"
  4. Click "OK"
  5. Exit once complete, then restart
