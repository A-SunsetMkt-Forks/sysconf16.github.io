Configuring Microsoft Edge on Windows

Preparatory Tasks

Collecting Configs
  1. Right-click the following links, "Save link as", and save on the desktop
    • my-ublock-backup.txt
    • SponsorBlockConfig.json
First Launch
  1. Start Menu > click "Microsoft Edge"
  2. Click "Start without your data"
  3. Click "Don't allow", then "Confirm and continue"
  4. Click "Next", then "Finish"
  5. Click "Reject All"
  6. Close all tabs except the New Tab Page
  7. Hit "Alt + F" on your keyboard, then click "Settings"
  8. Click "About Microsoft Edge", update if necessary, then restart the browser
  9. Close the browser once it reopens
Running the Configurator
  1. Visit following link on the target system and download the script. Extract to the desktop using 7-Zip
    • Edge Configurator | Sets the Optimal Settings for Microsoft Edge on Windows
  2. Run "runme.cmd"
  3. Start Menu > click "Microsoft Edge"
  4. Hit "Alt + F" on your keyboard, then click "Settings"

Settings

Privacy, search and services
  1. Security > turn off "Microsoft Defender SmartScreen"
  2. Turn off "Typosquatting Checker"
  3. Turn off "Turn on site safety services to get more info about the sites you visit"
  4. Services > click "Address bar and search"
  5. Click "Manage search engines"
  6. Click the 3 dots next to "Google" and click "Make default"
  7. System and performance > turn off "Startup boost"
  8. Change "Turn on efficiency mode when" to "Always"
Flags
  1. Type edge://flags into the address bar
  2. Type "Hardware Media Key Handling" into the Search box
  3. Change "Default" to "Disabled" and click "Restart"

Extensions

  1. Visit the following links and click "Add to Chrome", then "Add extension"
    • uBlock Origin | Wide-Spectrum Blocker
    • SponsorBlock | YouTube Sponsorship Segment Skipper
  2. Right-click the uBlock Origin icon > click "Extension options"
  3. Click "Restore from file", navigate to and select "my-ublock-backup.txt", click "Allow" if prompted then click "OK"
  4. Re-open Microsoft Edge, right-click the uBlock Origin icon > click "Extension options"
  5. Filter lists > click "Update now"
  6. Right-click the SponsorBlock icon > click "Extension options"
  7. Backup/Restore > click "Load from file", navigate to and select "SponsorBlockConfig.json", click "Allow" if prompted then click "OK"
  8. Exit
