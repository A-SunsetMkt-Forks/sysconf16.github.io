Building Downtify

Preparatory Tasks

  1. Download and install the installers from the following links
    • .NET Framework 4.6.1 Developer Pack Installer
    • .NET 6.0 SDK (v6.0.200) - Windows x64 Installer
  2. Save and extract the following archive
    • master.zip | Downtify Source

Building

  1. Start Menu > Windows System > right-click "Command Prompt" > More > click "Run as administrator"
  2. Type the following commands
    • cd filepathoftheextractedmasterfolder
    • dotnet.exe build
  3. Once complete, exit and navigate to the "bin" folder to find the executable
