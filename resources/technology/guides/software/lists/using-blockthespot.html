Using BlockTheSpot

Preparatory Tasks

  1. Uninstall any installations of Spotify present on the target system
  2. Download and install Spotify from the following link
    • Spotify | Win32 Spotify Installer
  3. Download the following script to your downloads folder
    • BlockTheSpot.bat | Windows Spotify Client Ad-Blocker

Installation

  1. Navigate to your downloads folder and right-click "BlockTheSpot.bat", then click "Run as administrator"
  2. At the "Uninstall Spotify Windows Store edition if it exists" prompt, type "n" then hit the enter key
  3. At the "Remove ad placeholder and upgrade button" prompt, type "y" then hit the enter key
  4. Wait for completion then exit

Info

  • BlockTheSpot will stop working after every Spotify update
  • Repeat the steps above to restore adblocking
