Using BlockTheSpot

Preparatory Tasks

  1. Download and install Spotify from the following link
    • Spotify | Win32 Spotify Installer

Installation

  1. Start Menu > Windows PowerShell > click "Windows PowerShell"
  2. Copy and paste [Net.ServicePointManager]::SecurityProtocol = [Net.SecurityProtocolType]::Tls12; Invoke-Expression "& { $(Invoke-WebRequest -UseBasicParsing 'https://raw.githubusercontent.com/mrpond/BlockTheSpot/master/install.ps1') } -UninstallSpotifyStoreEdition -RemoveAdPlaceholder" into the PowerShell window, then hit the enter key
  3. Wait for completion then exit

Info

  • BlockTheSpot will stop working after every Spotify update
  • Repeat the steps above to restore adblocking
