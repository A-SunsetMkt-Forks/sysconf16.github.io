Sideloading on iOS & iPadOS with TrollStore (<16.6.1)

Preparatory Tasks

Enabling Developer Mode
  1. On your iOS or iPadOS device, open Settings
  2. Privacy & Security > Developer Mode
  3. Toggle the switch "On" and follow the instructions to restart
Collecting Tools
  1. Download and install Sideloadly to a computer from the following link
    • Sideloadly | .ipa Installer
  2. If you're using Windows, uninstall any installations of iTunes present on the target system, then download and install iTunes using the following link
    • iTunes | Win32 iTunes Installer
  3. Download the .ipa file for the app you'd like to install to your device
    • Spotilife | iOS Spotify Client with Premium Features (try 8.6.42 first)
  4. Remove any installations of an app already present on the target device before attempting to install a sideloaded copy
  5. Ensure you have the Tips app installed on your device
Creating a Temporary Apple ID (optional)
  1. On your iOS or iPadOS device, open Settings
  2. Mail > Accounts > tap "Add Account", then tap "iCloud"
  3. Tap "Create a new Apple ID"

Installing and using TrollStore

Installation
  1. Visit this link and follow the steps for your device's corresponding iOS/iPadOS version under the arm64e list
Installing Apps
  1. On your device, locate the previously downloaded .ipa file in the Files app
  2. Tap the .ipa file then the Share icon
  3. Tap "TrollStore"
  4. Tap "Install" and wait for completion

Info

  • If premium features are not enabled with Spotify 8.6.42, tap the cog in the top-right of the app from the home screen, "Spotilife" > turn on "Enabled", "Apply Changes (Close Spotify) and re-open
  • Ensure not to update iOS/iPadOS to retain TrollStore functionality
