Optimising Windows 10

Preparatory Tasks

Disabling UAC
  1. Hit "Win + S" on your keyboard to open Search
  2. Type "UAC" and click "Change User Account Control settings"
  3. Slide the blue slider to "Never notify" and click "OK"
  4. Click "Yes"
Disabling Real-Time Protection
  1. Start Menu > Windows Security > Virus & threat protection > "Manage settings" under "Virus & threat protection settings"
  2. Turn off "Real-time protection"
  3. Exit
Collecting Tools
  1. On the desktop of the target system, create 4 folders named "configs", "tools", "scripts" and "installers"
  2. Right-click the following link, "Save link as", and save inside the "configs" folder
    • ooshutup10.cfg
  3. Visit the following links and download the portable executable archives. Extract and save all inside the "tools" folder
    • Privatezilla | FOSS Privacy Settings Changer
    • CCleaner | System Cleaner
    • PrivaZer | System Cleaner
    • Defender Control | Microsoft Defender Disabler
    • ShutUp10++ | Privacy Settings Changer
    • WPD | Privacy Settings Changer
    • InSpectre | Meltdown & Spectre Patch Manager
  4. Visit the following links and download the executables. Save the files inside the "installers" folder
    • 7-Zip | FOSS Archiver
    • JPEGView | FOSS Image Viewer
    • MPC-HC | FOSS Media Player
    • SumatraPDF | FOSS PDF Viewer
  5. Visit the following link and download the archive. Extract and place the "scripts" folder in the same location as "Privatezilla.exe"
    • packages.zip | Accompanying Scripts for Privatezilla
  6. Right-click the following link, "Save link as", and save inside the "scripts" folder
    • privacy-script.bat
  7. Visit the following link then download and extract the script to the "scripts" folder
    • JPEGView and SumatraPDF Configurator | Sets the Optimal Settings for JPEGView and SumatraPDF

Managing Telemetry, Data Collection & Services

privacy.sexy Script
  1. In the "scripts" folder, right-click "privacy-script.bat", then click "Run as administrator"
  2. Restart once the script is complete
Defender Control
  1. Open "dControl" from the "tools" folder
  2. Click "Disable Windows Defender"
  3. Manually disable Tamper protection if prompted
  4. Exit
ShutUp10
  1. Open "OOSU10" from the "tools" folder
  2. File > Import settings... > select "ooshutup10.cfg" from the "configs" folder > click "OK"
  3. Exit > click "OK" > click "Close"
Privatezilla
  1. Open "Privatezilla" from the "tools" folder
  2. Under the "Scripts" tab, check the following:
    • Block Telemetry IPs via Windows Firewall
    • Disable Services
    • Remove default apps (Provisioned)
    • Remove default apps (Recommended)
    • Remove OneDrive
    • Run Windows10 Debloater (Community)
  3. Click "Apply selected"
  4. Click "OK" 4 times
  5. If the script becomes stuck at "Waiting for explorer to complete loading", close the Powershell window manually, then click "OK"
  6. Click "Yes" 4 times, then "No" 3 times
  7. Click "OK"
  8. Under the "Settings" tab, check the following:
    • Privacy
    • Cortana
    • Remove all built-in apps
    • App permissions, then uncheck "Disable app notifications" and "Disable app access to camera, microphone, notifications, account info, documents, pictures, videos, file system"
    • Disable Windows update sharing
    • Gaming
    • Windows Defender
    • Microsoft Edge, then uncheck "Block Installation of New Microsoft Edge
    • Disable DRM in Windows Media Player
  9. Click "Apply selected"
  10. Exit
WPD
  1. Open "WPD" from the "tools" folder
  2. Click "Privacy"
  3. Turn off the following:
    • Everything under "Local group policy"
    • Everything under "Services"
    • Everything under "Task Scheduler" except "Consolidator" and "Device Census"
    • Everything under advanced "Local group policy" except "Allow Clipboard History", "Use OneDrive for file storage" and "Let Windows apps access account information, the camera, the microphone, notifications"
    • Everything under advanced "Services" except "Windows Push Notifications User Services"
    • Everything under "Miscellaneous"
  4. Leave "Windows Update" enabled
  5. Click the "Blocker" tab
  6. Click "Telemetry" under "Rule sets"
  7. Exit
PrivaZer
  1. Open "PrivaZer" from the "tools" folder
  2. Click "Cancel"
  3. Click "Advanced Options"
  4. Click the "Memory" tab
  5. Click "Stop" under "Wsearch Indexing"
  6. Exit
CCleaner
  1. Open "CCleaner64" from the "tools" folder
  2. Click "Tools", then "Startup"
  3. Click "SecurityHealth", then "Delete"
  4. Click "Scheduled Tasks", then "CCleanerSkipUAC - User", then "Delete", then "OK"
  5. Click "MicrosoftEdgeUpdateTaskMachineCore", then "Disable"
  6. Click "MicrosoftEdgeUpdateTaskMachineUA", then "Disable"
  7. Click "Windows Services", then check "Advanced Mode"
  8. Disable all except the following:
    • Apple Mobile Device Service (if present)
    • Background Tasks Infrastructure Service
    • Base Filtering Engine
    • Core Messaging
    • DCOM Server Process Launcher
    • DNS Client
    • Group Policy Client
    • Local Session Manager
    • Microsoft Office Click-to-Run Service (if present)
    • NVIDIA Display Container LS (if present)
    • Print Spooler
    • Remote Procedure Call (RPC)
    • RPC Endpoint Mapper
    • Security Center
    • Software Protection
    • System Events Broker
    • System Guard Runtime Monitor Broker
    • Task Scheduler
    • Windows Audio
    • Windows Audio Endpoint Builder
    • Windows Defender Firewall
    • WLAN AutoConfig
    • WpnUserService (if present)
  9. Exit
InSpectre (on Intel Systems Only)
  1. Right-click "InSpectre" from the "tools" folder and click "Run as administrator"
  2. Click "Disable Meltdown Protection" and "Disable Spectre Protection"
  3. Exit

Configuring Windows

  • Restart before beginning the following steps
Start Menu
  1. Right-click and unpin all tiles
Taskbar
  1. Right-click > Search > click "Show search icon"
  2. Right-click > uncheck "Show Task View button"
  3. Start Menu > right-click "Microsoft Edge" > More > click "Pin to taskbar"
  4. Start Menu > Windows System > right-click File Explorer > More > click "Pin to taskbar"
Explorer
  1. Hit "Win + E" on your keyboard to open an explorer window
  2. File > click "Change folder and search options"
  3. Click "Quick Access" and change to "This PC"
  4. Uncheck "Show frequently used folders in Quick access"
Settings
  1. Start Menu > click "Settings"
  2. Click "System" and change the following settings:
    • Sound > click "Sound Control Panel"
    • Right-click the default playback device > Properties > Enhancements > check "Disable all enhancements" > OK
    • Right-click the default recording device > Properties > Enhancements > check "Disable all enhancements" > OK
    • Communications > click "Do nothing"
    • Click "OK"
    • Hit "Win + Shift + S" on your keyboard to open Snip & Sketch, then hit the ESC key
    • Notifications & actions > Edit your quick actions > unpin all toggles on the Action Center window > click "Done"
    • Uncheck "Show reminders and incoming VoIP calls on the lock screen" and "Allow notifications to play sounds"
    • Uncheck all apps under "Get notifications from these apps", except "Snip & Sketch"
    • Focus assist > uncheck all "Automatic rules"
    • Click "Priority only" > click "Customise your priority list" > uncheck all "Calls, texts and reminders" options > uncheck all "People" options
    • Click and remove all "Apps" except "Snip & Sketch"
    • Power & sleep > change "When plugged in, turn off after" and "When plugged in, PC goes to sleep after" to "5 minutes"
    • Storage > turn off "Storage Sense"
    • Tablet > change "When I sign in" to "Never use tablet mode"
    • Multi-tasking > change "Pressing Alt + Tab shows" to "Open windows only"
    • Shared experiences > turn off "Share across devices"
    • Clipboard > turn on "Clipboard history"
    • About > click "Advanced system settings" > click "Advanced"
    • Click "Settings", then select "Adjust for best performance"
    • Check the "Smooth edges of screen fonts" and "Show thumbnails instead of icons" options, then click "OK"
    • Click the arrow in the top left to return
  3. Click "Devices" and change the following settings:
    • Bluetooth & other devices > turn off "Bluetooth" if not in use
    • Uncheck "Show notifications to connect using Swift Pair"
    • Printers & scanners > click "Fax" and "Remove device"
    • Typing > turn off "Autocorrect misspelt words"
    • AutoPlay > turn off "Use AutoPlay for all media and devices"
    • USB > uncheck "Notify me if there are issues connecting to USB devices"
    • Click the arrow in the top left to return
  4. Click "Network & Internet" and change the following settings:
    • WiFi > turn on "Use random hardware addresses"
    • Click the arrow in the top left to return
  5. Click "Personalisation" and change the following settings:
    • Colours > click "Choose your colour" > change to "Light" or "Dark" depending on preference
    • Turn off "Transparency effects"
    • Start > turn off "Show recently added apps"
    • Click "Choose which folders appear on Start" and turn off all options
    • Taskbar > turn on "Use small taskbar buttons"
    • Turn off "Replace Command Prompt with Windows Powershell in the menu when I right-click the start button or press Windows logo key + X"
    • Turn off "Show taskbar on all displays"
    • Click "Turn system icons on or off" and turn off "Microphone", "Location" and "Input Indicator"
    • Click the arrow in the top left to return
    • Click "Select which icons appear on the taskbar" and turn off "Microphone" and "Meet Now"
    • Click the arrow in the top left to return
  6. Click "Apps" and change the following settings:
    • Apps & features > click "Optional features" > click and uninstall "Windows Hello Face", "Windows Media Player" and "WordPad"
    • Click the arrow in the top left to return
  7. Click "Gaming" and change the following settings:
    • Xbox Game Bar > uncheck "Open Xbox Game Bar using this button on a controller."
  8. Click "Ease of Access" and change the following settings:
    • Mouse Pointer > click the far right option under "Change pointer colour", then "Pick a custom pointer colour", choose black then click "Done"
    • Uncheck "Show visual feedback around the touch points when I touch the screen"
    • Narrator > uncheck "Allow the shortcut key to start Narrator"
    • Keyboard > turn off "Allow the shortcut key to start Sticky Keys, Toggle Keys, Filter Keys"
    • Uncheck "Show a warning message when turning on Sticky Keys, Toggle Keys or Filter Keys from the keyboard"
    • Uncheck "Make a sound when turning Sticky Keys, Toggle Keys or Filter Keys on or off from the keyboard"
    • Click the arrow in the top left to return
  9. Click "Search" and change the following settings:
    • Permissions & History > change "SafeSearch to "Off"
    • Turn off "Search history on this device"
  10. Exit
Task Manager
  1. Right-click the Taskbar > click "Task Manager"
  2. Click "More details"
  3. Processes > right-click "Name" > uncheck "Status", "Power usage", "Power usage trend" > check "GPU"
  4. Performance > click "GPU 0" > right-click the graph space > "Change graph to" > click "Single engine"
  5. App history > right-click "Name" > uncheck "Metered network" and "Tile updates"
  6. Startup > right-click "Name" > uncheck "Publisher" and "Start-up impact"
  7. Select any options present in the list, then "Disable"
  8. Users > right-click "Name" > uncheck "Status" > check "GPU"
  9. Exit
Control Panel
  1. Hit "Win + S" on your keyboard to open Search
  2. Type "Control Panel" and click "Control Panel"
  3. Programs > click "Turn Windows Features on or off"
  4. Uncheck "Remote Differential Compression API Support" and click "OK"
  5. Click "Close"
System Configuration
  1. Hit "Win + S" on your keyboard to open Search
  2. Type "msconfig" and click "System Configuration"
  3. Boot > check "No GUI boot" > click "OK" > click "Exit without restart"
Internet Properties
  1. Hit "Win + S" on your keyboard to open Search
  2. Type "inetcpl.cpl" and click "OK"
  3. Security > click "Custom level..."
  4. Miscellaneous > change "Launching applications and unsafe files" to "Enabled (not secure)", then click "OK" and "Yes"
  5. Click "OK"
Notepad
  1. Start Menu > Windows Accessories > click "Notepad"
  2. View > uncheck "Status Bar"
  3. Format > check "Word Wrap"
  4. Exit
Installing Basic Applications
  1. Navigate to the "installers" folder on the desktop
  2. Install each application
Configuring Basic Applications
  1. For 7-Zip
    1. Start Menu > 7-Zip > right-click "7-Zip File Manager" > More > Run as administrator
    2. Tools > System > Associate 7-Zip with: > click the "+" above "All users"
    3. Click "CompressedFolder" next to "zip"
    4. Click "CABFolder" next to "cab"
    5. 7-Zip > uncheck "Cascaded context menu"
    6. Uncheck all "Context menu items:" except "Extract files", "Add to .7z" and "Add to .zip"
    7. Click "OK" and exit
  2. For JPEGView & SumatraPDF:
    1. Navigate to the "scripts" folder on the desktop
    2. Run the jpegview-and-sumatrapdf-configurator-main "runme.cmd"
    3. Exit
  3. For MPC-HC
    1. Start Menu > MPC-HC x64 > right-click "MPC-HC x64" > More > Run as administrator
    2. View > Uncheck "Status"
    3. View > check "Hide Menu"
    4. View > Options
    5. Under "Player", uncheck "Keep history of recently opened files" and "Remember last playlist"
    6. Under "Formats", click the green checkmark under "File extensions"
    7. Uncheck "Use the format specific icons" and "File(s)" under "Explorer Context Icons"
    8. Under "Advanced" change "EnableCrashReporter" to "False"
    9. Exit
Setting Default Apps
  1. Start Menu > SumatraPDF > click the handburger menu in the top left > Settings > click "Options"
  2. Click "Make SumatraPDF the default PDF viewer" and exit
  3. Start Menu > JPEGView > click "Cancel" and right-click the black space > Settings/Admin > click "Set as the default viewer"
  4. Exit
  5. Start Menu > Settings > Apps > Default Apps
  6. Click "Paint" under "Photo viewer", then click "JPEGView"
  7. Click "Choose a default" under "Web browser", then click "Microsoft Edge"
  8. Click "Choose a default" under "Music player", then click "MPC-HC x64"
  9. Click "Choose a default" under "Video player", then click "MPC-HC x64"
  10. Click "Choose default applications by file type", find ".pdf", click the item listed then click "SumatraPDF"
  11. Exit
