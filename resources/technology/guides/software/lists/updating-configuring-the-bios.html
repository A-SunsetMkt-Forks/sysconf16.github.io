Updating & Configuring the BIOS

Update

  1. Download the latest BIOS revision for your motherboard from the manufacturer's website
  2. Format a USB stick with the FAT32 file system
  3. Extract the BIOS image from the downloaded archive
  4. Copy the BIOS image to the root of the USB stick (if the drive letter is "D", then D:\, i.e. not in a folder)
  5. Insert the USB stick into one of the rear USB 2.0 ports on the motherboard
  6. Press the power button and enter the BIOS by repeatedly pressing the BIOS key. See here for a list of BIOS keys
  7. Enter the BIOS flash/update utility for your motherboard
  8. Select the file from the USB stick and wait for the BIOS to update
  9. The system may power cycle multiple times and this is normal

Configuration

  1. Press the power button and enter the BIOS by repeatedly pressing the BIOS key. See here for a list of BIOS keys
  2. Find and enable the "XMP" profile for your memory
  3. Find and enable "UEFI Boot" and disable "Legacy Boot"
  4. Find and enable "PXE/network/LAN boot"
  5. Find and enable "Resizable Bar"
  6. Find and enable "4G Decoding"
  7. If there's an option to specify OS type (e.g., on some MSI and ASUS motherboards), select "Windows 10 WHQL"
  8. Disable "CSM Compatibility"
  9. Disable virtualisation, unless you'll be running virtual machines, by disabling "Intel Virtualisation Tech (VT-x)" for Intel and "SVM" for AMD
