Using SpotX

Preparatory Tasks

  1. Download and install Spotify from the following link
    • Spotify | Win32 Spotify Installer

Installation

  1. Start Menu > Windows PowerShell > click "Windows PowerShell"
  2. Copy and paste [Net.ServicePointManager]::SecurityProtocol = [Net.SecurityProtocolType]::Tls12; iex "& { $((iwr -useb 'https://raw.githubusercontent.com/amd64fox/SpotX/main/Install.ps1').Content) } -confirm_spoti_recomended_over -confirm_uninstall_ms_spoti -podcasts_off -block_update_off -cache_on -start_spoti" into the PowerShell window, then hit the enter key
  3. Wait for completion then exit

Info

  • SpotX will stop working after every Spotify update
  • Repeat the steps above to restore adblocking
