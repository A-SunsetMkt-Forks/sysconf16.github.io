PC Components

Processor (CPU)

  • Brain of the computer, capable of performing all types of calculations
  • Found with a variety of sockets depending on manufacturer
  • Can come with or without an integrated graphics processor (iGPU)
  • CPUs without an iGPU require an additional graphics card

CPU Cooler

  • Cools the processor (CPU)
  • Can be either air or liquid
  • Liquid coolers handle higher TDP CPUs, but lack the longevity and reliability of air coolers

Thermal Interface Material (TIM)

  • Conducts heat between the cold plate of the CPU cooler and the integrated heat spread (IHS) of the CPU

Memory (RAM)

  • Volatile, working memory which contains active processes and application data
  • Higher clockspeed and tighter timings = better performance

Motherboard

  • Allows communication between all components
  • Must be compatible with the processor and memory of choice
  • Found in a variety of sizes (micro-ATX, ATX, extended-ATX, ITX, etc)

Storage

  • Non-volatile persistent storage
  • Found as hard disk drives (HDDs) and solid state drives (SSDs)

Graphics Card (GPU)

  • Perform specific graphical data related calculations
  • Requires a PCI-E x16 slot and often supplemental power from the PSU

Power Supply (PSU)

  • Supplies power to all components
  • Found in a variety of sizes (ATX, SFX, TFX)

Case

  • Houses all components
  • Found in a variety of sizes (micro-ATX, ATX, extended-ATX, ITX, etc)

Expansion Cards

  • Add-on components for additional functionality, e.g. Wi-Fi or Bluetooth

Operating System (OS)

  • Software that provides a graphical user interface (GUI) for use
