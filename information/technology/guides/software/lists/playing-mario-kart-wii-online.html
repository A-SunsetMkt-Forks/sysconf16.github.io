Playing Mario Kart Wii Online

Prerequisites

  • Working computer
  • Bluetooth compatible controller

Setting-up Dolphin

Collecting Files

  2. Download the latest development build for your operating system, e.g., "Windows x64" for Windows users or "macOS (ARM/Intel Universal) for Mac users
  3. Visit either of the following links to download the Dolphin configuration files and Mario Kart Wii ROM, depending on the performance of your computer

    • Low-Spec Pack via Filen | for low-end computers, including laptops

    • 1440p Upscaled Pack via Filen | for high-end computers

On Windows
  1. Extract the "dolphin-master-5.0-XXXXX-x64.7z" archive to a location you'd like for the emulator to be stored
  2. Hit "Win + E" on your keyboard to open an Explorer window
  3. Navigate to "Documents"
  4. Create a folder called "Dolphin Emulator"
  5. Move all of the folders located within the previously extracted archive containing the configuration files into the newly created "Dolphin Emulator" folder
  6. Exit
On macOS
  1. Open a new Finder window
  2. With the Finder window active, click the "Shift", "Command" and "> ." keys simultaneously to enable hidden files
  3. Open the "dolphin-master-5.0-XXXXX-universal.dmg" installer and copy "Dolphin" to "Applications" in the Finder window
  4. Navigate to "Macintosh HD" > "Users" > "[Your Username]" > "Library" > "Application Support"
  5. Create a new folder called "Dolphin"
  6. Move all of the folders located within the previously extracted archive containing the configuration files into the newly created "Dolphin" folder
  7. Click the "Shift", "Command" and "> ." keys simultaneously to disable hidden files
  8. Exit
  9. Open Dolphin
  10. Click "Options" > "Graphics Settings" then change the "backend" from "Vulkan" to "Metal"
  11. Exit

Using Dolphin

Creating a Mii
  1. Open Dolphin
  2. Click "Tools", then "Load Wii System Menu 4.3U" to open the Wii Menu
  3. Click the newly opened window, then open the Mii Channel to start
  4. Once your Mii has been created, click the "esc" key to exit
  5. Close Dolphin
Mapping Controller Buttons
  1. Connect your controller via Bluetooth
  2. Open Dolphin
  3. Click "Options", then "Controller Settings"
  4. Click the "Wii Remote 1" drop-down under "Wii Remotes" and set to "None"
  5. Click the "Port 1" drop-down under "GameCube Controllers" and set to "Standard Controller"
  6. Click "Configure" next to the "Port 1" option
  7. Click the "Device" drop-down to change "Keyboard & Mouse" to your connected bluetooth controller"
  8. Click "Clear" under the Reset heading to clear all mappings
  9. Map all controls, setting "Z" as R1 / RB / R and "D-Pad Up" as L1 / LB / L for tricks and wheelies
  10. Click "Close", then "Close" again
Joining an Online Match for the First Time
  1. Click "Options" > "Paths" then click the "Add" button
  2. Navigate to the "Dolphin" or "Dolphin Emulator" folder depending on your operating system, then double-click the "Software" folder and set it as a path
  3. Close the window then double-click "Mario Kart Wii" to launch the game
  4. Follow the game instructions to create a licence using your Mii
  5. Access "Nintendo WFC" from the main game menu and approve access
  6. Launch a "Regional" game at the same time as another person to be automatically connected
  7. Click the "esc" key to exit once finished
