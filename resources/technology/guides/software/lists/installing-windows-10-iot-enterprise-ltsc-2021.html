Installing Windows 10 IoT Enterprise LTSC 2021

Prerequisites
  • Working computer with an internet connection
  • Blank 8GB+ USB stick
  • All existing data from the target computer backed up onto an external drive or cloud storage

Creating Installation Media

Flashing the ISO Image
  1. Using a working Windows computer, visit this link to download the x64-based ISO image of the Windows setup that matches your region
  2. Download the portable Rufus application here
  3. Insert the USB stick into the computer and ensure that it is empty
  4. Run Rufus and choose the USB stick
  5. Click "SELECT" and navigate to the downloaded ISO image
  6. Click "START" and wait for the process to complete
Collecting Installers & Tools
  1. In the root of the USB drive, create a folder named "files"
  2. In "files" folder, create 3 further folders named "drivers", "installers" and "scripts"
  3. Visit the following links relevant to your hardware and download the executables. Save the files inside the "drivers" folder
    • NVCleanstall | for Nvidia GPUs
    • AMD Drivers and Support | for AMD GPUs & Chipsets
    • Download Drivers & Software | for Intel iGPUs & Chipsets
  4. Visit the following links and download the executables. Save the files inside the "installers" folder
    • 7-Zip | FOSS Archiver
    • JPEGView | FOSS Image Viewer
    • MPC-HC | FOSS Media Player
    • SumatraPDF | FOSS PDF Viewer
  5. Visit the following links and download the scripts. Save the files inside the "scripts" folder, then extract using 7-Zip
    • SYSCONF16 LTSC 2021 Activator | Converts Enterprise to IoT Enterprise & Activates via HWID
    • SYSCONF16 LTSC 2021 Snip Installer | Installs the Packages Required to Restore Clipboard History & Snip & Sketch
    • SYSCONF16 LTSC 2021 JPEGView and SumatraPDF Configurator | Sets the Optimal Settings for JPEGView and SumatraPDF

Installation

Booting from USB
  1. Turn off the target computer and insert the USB stick
  2. Push the power button while repeatedly pressing the boot menu key. A list for several manufacturers can be found here
  3. Select the UEFI option for your USB drive from the list
Installing to Disk
  1. Continue through the wizard, then click "Custom: Install Windows only (advanced)"
  2. Select and delete all partitions on the desired boot drive so that only a single "Drive X Unallocated Space" is listed
  3. Select "Drive X Unallocated Space" and click "Next"
  4. Wait for the installation to complete
Out-of-Box-Experience (OOBE)
  1. Disconnect your ethernet cable if using a wired connection
  2. Continue through the wizard, then click "I don't have internet" when prompted to create an internet connection and "Continue with limited setup"
  3. Click "No" for all privacy settings and choose "Send Required diagnostic data"
  4. At the desktop, re-connect your ethernet cable or connect to Wi-Fi

Core Functionality & Configuration

Installing Hardware Drivers
  1. Navigate to the "drivers" folder on the USB stick
  2. Install each driver package
  3. Restart after the final package has been installed
Activating Windows
  1. Navigate to the "scripts" folder on the USB stick
  2. Run the SYSCONF16 LTSC 2021 Activator "runme.cmd"
  3. Click "More info" on the SmartScreen prompt, then "Run anyway"
  4. Click "Yes" on the UAC prompt
Restoring Clipboard History & Snip & Sketch
  1. Navigate to the "scripts" folder on the USB stick
  2. Run the SYSCONF16 LTSC 2021 Snip Installer "runme.bat"
  3. Click "More info" on the SmartScreen prompt, then "Run anyway"
  4. Click "Yes" on the UAC prompt
Running Windows Update
  1. Start Menu > Settings > Update & Security > Windows Update
  2. Check for and install all available updates
  3. Restart
Installing Basic Applications
  1. Navigate to the "installers" folder on the USB stick
  2. Install each application
Configuring Basic Applications
  1. For 7-Zip
    1. Start Menu > 7-Zip > right-click "7-Zip File Manager" > More > Run as administrator
    2. Tools > System > Associate 7-Zip with: > click the "+" above "All users"
    3. Click "CompressedFolder" next to "zip"
    4. Click "CABFolder" next to "cab"
    5. 7-Zip > uncheck "Cascaded context menu"
    6. Uncheck all "Context menu items:" except "Extract files", "Add to .7z" and "Add to .zip"
    7. Click "OK" and exit
  2. For JPEGView & SumatraPDF:
    1. Navigate to the "scripts" folder on the USB stick
    2. Run the SYSCONF16 LTSC 2021 JPEGView and SumatraPDF Configurator "runme.cmd"
    3. Click "More info" on the SmartScreen prompt, then "Run anyway"
    4. Click "Yes" on the UAC prompt
  3. For MPC-HC
    1. Start Menu > MPC-HC x64 > right-click "MPC-HC x64" > More > Run as administrator
    2. View > Uncheck "Status
    3. View > Options
    4. Under "Player", uncheck "Keep history of recently opened files" and "Remember last playlist"
    5. Under "Formats", click the green checkmark under "File extensions"
    6. Uncheck "Use the format specific icons" and "File(s)" under "Explorer Context Icons"
    7. Under "Advanced" change "EnableCrashReporter" to "False"
    8. Exit
  4. Setting Default Apps (for applications that do not show as options, try opening a file using it)
    1. Start Menu > Settings > Apps > Default Apps
    2. Click "Paint" under "Photo viewer", then click "JPEGView"
    3. Click "Choose a default" under "Web browser", then click "Microsoft Edge"
    4. Click "Choose default applications by file type", find ".pdf", click the item listed then click "SumatraPDF"
    5. Exit
    6. Start Menu > right-click "Microsoft Edge" > More > click "Pin to taskbar"
    7. Start Menu > Windows System > right-click File Explorer > More > click "Pin to taskbar"
