Extracting Archives with Passwords on Windows & macOS

Preparatory Tasks

On Windows
  1. Download and install 7-Zip to the target machine using the following link
    • 7-Zip | FOSS Archiver
  2. Configure 7-Zip using the following steps
    1. Start Menu > 7-Zip > right-click "7-Zip File Manager" > More > Run as administrator
    2. Tools > System > Associate 7-Zip with: > click the "+" above "All users"
    3. Click "CompressedFolder" next to "zip"
    4. Click "CABFolder" next to "cab"
    5. 7-Zip > uncheck "Cascaded context menu"
    6. Uncheck all "Context menu items:" except "Extract files" and "Add to archive..."
    7. Click "OK" and exit
On macOS
  1. Download and install Keka to the target machine using the following link
    • Keka | macOS Archiver
  2. Configure Keka using the following steps
    1. Finder > Applications > click "Keka", click "Open" then click "Understood"
    2. Click "Keka" on the Menu Bar > click "Preferences"
    3. General > change "Default action:" to "Perform automatic action"
    4. General > uncheck "Enable Notification Centre integration"
    5. General > check "Close Keka when no windows are open"
    6. General > uncheck "Automatically check for updates"
    7. Appearance > uncheck "Use custom tasks counter in the Dock"
    8. Compression > change "Default format" to "7Z"
    9. Compression > change "Default method" to "Slowest, more compression"
    10. Compression > change "Save to location" to "Next to original file"
    11. Compression > change "Name of new files" to "Same as original file"
    12. Compression > check "Use parent folder name when compressing multiple files"
    13. Compression > change "Play a sound after a compression" to "None"
    14. Compression > check "Exclude Mac resource forks"
    15. Compression > uncheck "Show file in Finder after compression"
    16. Extraction > change "Extract to location" to "Next to original file"
    17. Extraction > change "Play a sound after an extraction" to "None"
    18. Extraction > check "Exclude Mac resource forks"
    19. Extraction > uncheck "Show content in Finder after extraction"
    20. Extraction > check "Remove bad password header structure extractions"
    21. Extraction > check "Rename the recognised extension from the subfolder"
    22. File Access > click "+", select "Macintosh HD" then click "Give access to Keka"
    23. Performance > change "Maximum simultaneous operations" to "1"
    24. Performance > change "Maximum simultaneous compressions" to "1"
    25. Performance > change "Maximum simultaneous extractions" to "1"
    26. Finder Extension > only check "Show Keka actions in Finder's contextual menu" and "Use "Open with Keka" as the primary item action"
    27. Exit

Extraction

On Windows
  1. Right-click the archive, then click "Extract files..."
  2. Enter the password into the "Password" field and click "OK"
On macOS
  1. Right-click the archive > Open With > click "Keka"
  2. Enter the password and click "Done"
