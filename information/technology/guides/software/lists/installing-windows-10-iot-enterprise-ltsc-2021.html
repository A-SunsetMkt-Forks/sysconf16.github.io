Installing Windows 10 IoT Enterprise LTSC 2021

Prerequisites

  • Computer with an internet connection
  • Blank 8GB+ USB stick
  • All existing data from the target computer backed up onto an external drive or cloud storage

Creating Installation Media

Flashing the ISO Image
  1. Using a working Windows computer, visit this link to download the 64-bit ISO en_gb image of the Windows installation media
  2. Download the portable Rufus application here
  3. Insert the USB stick into the computer and ensure that it is empty
  4. Run Rufus and choose the USB stick
  5. Click "SELECT" and navigate to the downloaded ISO image
  6. Ignore all "Windows User Experience" options and click "OK"
  7. Click "START" and wait for the process to complete
Collecting Installers & Tools
  1. In the root of the USB drive, create a folder named "files"
  2. In "files" folder, create 2 further folders named "drivers" and "scripts"
  3. Visit the following links relevant to your hardware and download the executables. Save the files inside the "drivers" folder
    • NVCleanstall | for Nvidia GPUs
    • AMD Drivers and Support | for AMD GPUs & Chipsets
    • Download Drivers & Software | for Intel iGPUs & Chipsets
  4. Visit the following links, then download and extract the scripts to the "scripts" folder
    • LTSC 2021 Activator | Converts Enterprise to IoT Enterprise & Activates via HWID
    • LTSC 2021 Snip Installer | Installs the Packages Required to Restore Clipboard History & Snip & Sketch

Installation

Booting from USB
  1. Turn off the target computer and insert the USB stick
  2. Push the power button while repeatedly pressing the boot menu key. A list for several manufacturers can be found here
  3. Select the UEFI option for your USB drive from the list
Installing to Disk
  1. Continue through the wizard, then click "Custom: Install Windows only (advanced)"
  2. Select and delete all partitions on the desired boot drive so that only a single "Drive X Unallocated Space" is listed
  3. Select "Drive X Unallocated Space" and click "Next"
  4. Wait for the installation to complete
Out-of-Box-Experience (OOBE)
  1. Disconnect your ethernet cable if using a wired connection
  2. Continue through the wizard, then click "I don't have internet" when prompted to create an internet connection and "Continue with limited setup"
  3. Click "No" for all privacy settings and choose "Send Required diagnostic data"
  4. At the desktop, re-connect your ethernet cable or connect to Wi-Fi

Core Functionality & Configuration

Installing Hardware Drivers
  1. Navigate to the "drivers" folder on the USB stick
  2. Install each driver package
  3. Restart after the final package has been installed
Activating Windows
  1. Navigate to the "scripts" folder on the USB stick
  2. Copy the "ltsc-2021-activator-main" folder to the desktop
  3. Run the ltsc-2021-activator-main "runme.cmd"
  4. Click "More info" on the SmartScreen prompt, then "Run anyway"
  5. Click "Yes" on the UAC prompt
Restoring Clipboard History & Snip & Sketch
  1. Navigate to the "scripts" folder on the USB stick
  2. Copy the "ltsc-2021-snip-installer-main" folder to the desktop
  3. Run the ltsc-2021-snip-installer-main "runme.bat"
  4. Click "More info" on the SmartScreen prompt, then "Run anyway"
  5. Click "Yes" on the UAC prompt
Running Windows Update
  1. Start Menu > Settings > Update & Security > Windows Update
  2. Check for and install all available updates
  3. Restart
